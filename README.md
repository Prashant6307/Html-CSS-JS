# HTML-CSS-JS Projects

This repository contains frontend web development projects created using HTML, CSS, and JavaScript. The projects focus on improving core web development skills such as responsive design, DOM manipulation, event handling, API integration, and UI development.

## Projects Included
- Bill Splitter App
- Meme Generator
- Dark Mode Toggle
- FAQ Accordion
- Counter App
- Weather App
- And more...

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Goal
To practice and improve frontend development skills by building real-world mini projects.
